About This App

This is a simple, AI-powered application designed to help you quickly extract information and get answers based on the content of your uploaded files.

Simply upload one or more PDF or text files, type your question, and our AI model will read and synthesize the information to provide a comprehensive response. It's fast, private, and efficient.

Technologies Used

  • Python: Main programming language.
  • Flask: Python web framework for the web application.
  • PyMuPDF: Python library for pdf text extraction.
  • HTML: For the frontend (client interactions).
  • CSS: For styling the HTML.
  • Mistral: LLM model for generating response.
